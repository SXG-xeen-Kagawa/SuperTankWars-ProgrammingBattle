# ゲームパッド操作方法

ゲームパッドで戦車を操作できます。  
戦車の重さや砲塔・装甲の配置によって挙動が大きく変わるため、試運転や動作検証に便利です。

※操作は少し難しいので、最初は「動かせたらOK」くらいの気持ちでお試しください。

---

## 使い方（コード側の設定）

ゲームパッドで操作したい戦車のスクリプト（`ComPlayerBase` を継承したクラス）の `Update()` 内に、以下を記述してください。

```C#
SXG_TestPlayByGamepad();
```

検証が終わったら、コメントアウト（または削除）してください。

---

## 操作方法

| ボタン | 説明 |
| ---- | ---- |
| 左スティックY軸 | 左のキャタピラを前進・後退させます |
| 右スティックY軸 | 右のキャタピラを前進・後退させます |
| 左右スティックX軸（外に開く） | すべての砲塔を上に向けます（ピッチ角） |
| 左右スティックX軸（内に閉じる） | すべての砲塔を下に向けます（ピッチ角） |
| Lボタン | すべての砲塔を左方向に回転させます |
| Rボタン | すべての砲塔を右方向に回転させます |
| Rトリガー | すべての砲塔の砲弾を発射します |

---

## 補足

- 「左右スティックX軸（外に開く）」は、左スティックを左側に、右スティックを右側に、同時に動かします。  
- 「左右スティックX軸（内に閉じる）」は、左スティックを右側に、右スティックを左側に、同時に動かします。
